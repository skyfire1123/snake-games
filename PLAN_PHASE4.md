## ⚡ 道具系统（Power-up System）— 最高优先级

### 核心设计原则
- 道具从地图上**随机生成**（非购买），增加游戏惊喜感
- 同时最多 **1 个**道具在场，避免道具堆叠破坏平衡
- 道具存在 **8 秒**，不捡取自动消失，避免卡位
- 道具**不分模式**（Classic/Endless/Challenge 均可使用）
- 道具效果**可叠加**（如 Shield + Slow 可同时生效）

---

### 道具清单（共 6 种）

| 道具名 | 图标色 | 效果 | 时长 | 视觉 |
|--------|--------|------|------|------|
| 🛡️ 护盾 Shield | 蓝色气泡 | 抵挡 **1 次**死亡（撞墙/撞身无效）| 永久（直到触发）| 蛇身外层蓝色光圈 |
| 🐢 减速 Slow | 紫色雪花 | 速度 **-50%**（interval ×2）| 5 秒 | 蛇身变半透明+冰晶粒子 |
| 👻 穿墙 Ghost | 白色幽灵 | 可**穿越自身**（穿墙模式暂时启用）| 3 秒 | 蛇身 50% 透明度 + 残影 |
| 🧲 磁铁 Magnet | 金色星形 | 食物被**吸引**向蛇头（半径 5 格）| 5 秒 | 蛇头金色光环 |
| 💰 双倍 Double | 红色双星 | 所有得分 **×2**| 10 秒 | HUD 显示 "×2" 高亮 |
| ✂️ 收缩 Shrink | 绿色剪刀 | 蛇身**减少 3 节**（尾巴消失）| 即时 | 尾部粒子爆散 |

---

### 道具生成规则

```
生成触发：每吃满 5 个食物 → 50% 概率生成 1 个随机道具
         + 每局**固定**在第 10 秒时若场上无道具则生成 1 个
道具在场上限：1 个（生成前检查，已存在则不生成）
存在时限：8 秒后自动消失（不触发效果）
不生成位置：蛇身占据的格子、食物所在的格子
```

**随机权重：**

| 道具 | 权重 |
|------|------|
| 护盾 | 20% |
| 减速 | 20% |
| 穿墙 | 15% |
| 磁铁 | 20% |
| 双倍 | 15% |
| 收缩 | 10% |

（收缩权重较低因为即时触发风险高，护盾/减速/磁铁权重较高因为体验正反馈强）

---

### 道具碰撞检测

道具节点为 `Area2D`，与蛇头 `Area2D` 碰撞后：
1. 触发对应效果（立即执行或启动计时器）
2. 播放收集音效 `powerup_collect.wav`
3. 粒子爆散效果
4. 从地图移除

### 技术实现

```gdscript
# snake.gd 扩展
const SKIN_PRESETS := {
    "neon_green": { "head": {...}, "body": {...}, "tail": {...} },
    "midnight_blue": { ... },
}

var _current_skin_id := "neon_green"
```

- 付费皮肤：保留独立渐变 sprite，支持颜色循环/脉冲动画

## ⚠️ 风险日志

| 风险 | 影响 | 应对 |
|------|------|------|
| 道具与食物重叠生成 | 玩家无法到达 | spawn 前检查 occupied_cells + active_food_positions |
| 护盾 + 穿墙同时激活导致无限不死 | 游戏机制崩坏 | 穿墙效果**不穿过蛇身**，仅穿网格边界 |
| 磁铁吸引导致食物移出边界 | 显示 BUG | 吸引范围限制在 5 格半径内，到边界停止 |
| 收缩导致蛇身长度 < 3 | 蛇消失 | min_length = 3（初始值），shrink 不突破此下限 |
| 道具过于稀有导致体验不到 | 玩家感受不到乐趣 | 确保每局**平均**生成 2–3 个道具（基于吃5食触发概率）|
| 低端机粒子性能下降 | 帧率降低 | 粒子数量与设备性能挂钩（可配置关闭）|
| 皮肤 sprite 增大安装包 | 包体超标 | 皮肤启用 ETC1 压缩，延迟加载（非启动预加载）|
